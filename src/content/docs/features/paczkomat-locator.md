## Overview

Paczkomat Locator helps you find the closest InPost parcel locker for any address in Poland. With over 24,700 Paczkomat locations indexed, the tool uses Nominatim geocoding to convert addresses into coordinates and returns nearby lockers sorted by distance. Select a Paczkomat directly when creating shipments to streamline your fulfillment workflow.

## Key Features

- **24,700+ locations** — full coverage of InPost Paczkomat network across Poland, updated regularly.
- **Address-based search** — enter a street address, city, or postal code and get the nearest lockers instantly.
- **Nominatim geocoding** — converts addresses to geographic coordinates for accurate distance calculations.
- **Point details** — view locker name, full address, type (Paczkomat, PaczkoPunkt, or Pop), and operational status.
- **Status indicators** — see whether a locker is operational, full, or temporarily out of service.
- **Shipping integration** — select a Paczkomat directly during label creation to pre-fill delivery details.

## How It Works

1. Open the **Paczkomat Locator** from the dashboard or the shipment creation form.
2. Enter an address, city name, or postal code in the search field.
3. Nominatim geocodes the input and the system finds Paczkomats within the search radius.
4. Browse results sorted by distance. Each entry shows the point name, address, type, and current status.
5. Click a result to see it on the map and view additional details.
6. Select **Use this Paczkomat** to attach it to your current shipment or save it as a preferred location.
