## Overview

Logistics centralizes your shipping operations across seven major Polish carriers. Connect InPost (Paczkomaty), DPD, DHL, GLS, Poczta Polska, and ORLEN Paczka, then compare rates, generate labels, and track every parcel from a single dashboard. Stop juggling multiple carrier portals and let EcomPilot handle fulfillment end to end.

## Key Features

- **7 carrier integrations** — InPost (Paczkomaty), DPD, DHL, GLS, Poczta Polska, and ORLEN Paczka connected via official APIs.
- **Rate comparison** — enter package dimensions and weight to see real-time pricing from all connected carriers side by side.
- **Label generation** — create shipping labels in one click, formatted to each carrier's specifications.
- **Unified shipment tracking** — monitor all parcels across carriers in a single tracking view with status updates.
- **Bulk shipping** — generate labels and dispatch multiple orders in batch to save time during peak periods.
- **Delivery notifications** — automatically notify buyers when their order ships and when it arrives.

## How It Works

1. Open **Logistics** from the dashboard or an order detail page.
2. Select the order(s) you want to ship.
3. Enter or confirm package dimensions and weight.
4. The system queries all connected carriers and displays available services with prices and estimated delivery times.
5. Select a carrier and service level. Click **Generate Label** to create the shipping label.
6. Print the label, hand off the parcel, and track delivery status in the **Shipments** tab.
7. Buyers receive automatic notifications at each tracking milestone.
